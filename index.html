Mission : Capture de l'Aérodrome de Carpiquet

Moment Historique

Bataille de Carpiquet (Normandie, juillet 1944)

  • Contexte : Après le débarquement en Normandie, les Alliés visent les aérodromes pour soutenir les opérations aériennes.
  • Capture : Opération Windsor lancée le 4 juillet par les Canadiens pour prendre l’aérodrome de Carpiquet près de Caen.
  • Combats : Soutien blindé et aérien, résistance allemande intense. L'aérodrome est capturé le 9 juillet.

Bienvenue dans la section Aide pour les événements

Ici, vous trouverez les solutions aux problèmes liés aux cartes, camouflages ou missions personnalisées.

Un problème avec le téléchargement des missions ?

  1. Carte corrompue
    • Retéléchargez la carte posant problème puis relancez le jeu.
    • La carte n’est peut-être plus à jour et ne peut plus être jouée.
  2. Erreur “Failed to load”
    • Relancez le jeu.
  3. Comment lancer une bataille personnalisée
    • Installez la carte via le lien proposé directement.
    • Lancez la carte via le lien de la mission.

Un problème avec le téléchargement des camouflages ?

  1. Comment installer un camouflage
    • Téléchargez un camouflage sur le Live War Thunder.
    • Une fois téléchargé, décompressez-le.
    • Placez le dossier décompressé dans le dossier UserSkins du jeu.
    • Lancez le jeu, sélectionnez votre véhicule, puis appliquez le camouflage.
    • Si le camouflage n’apparaît pas, cliquez sur la petite flèche circulaire dans l’interface.
  2. Comment afficher les camouflages
    • Allez dans les options (en haut à gauche du menu principal).
    • Dans la section “Customisation des véhicules”, activez les camouflages personnalisés.
  3. Modifier un camouflage stock ou téléchargé
    • Enregistrez-le depuis l’interface de customisation (icône de baguette magique).
    • Un message apparaît pour vous signaler que le camouflage a été créé.
    • Allez dans le fichier UserSkins pour le retrouver.
    • Sélectionnez le fichier créé, puis ouvrez-le dans un éditeur d’image (comme Photoshop).
    • Après modification, enregistrez dans le dossier UserSkins.
    • Félicitations, vous avez créé un camouflage personnalisé !
