Un problème avec le téléchargement des camouflages ?

  1. Comment installer le cammouflage

    Pour installer le cammouflage vous allez devoir Télécharger un cammouflage sur le Live War Thunder (mieux que vous passer sur un officiel)

    Une fois téléchargé, il faudra le déziper

    Une fois déziper il faudra aller dans les dossiers de War Thunder et le dossier "UserSkin"

    Une fois fait déplacer le fichier déziper du cammouflage à l'intérieur

    Lancer le jeu et aller sur votre véhicule le mettre, si vous ne trouvez pas le camouflage cliquer sur la petite flêche en rond comme indiqué

  2. Comment afficher les cammouflages

    Pour afficher les cammouflages vous devez aller dans les options (dans le menu en haut à gauche)

    Dans la section "Customisation" activer la customisation

  3. Modifier un cammouflage stock ou déjà télécharger

    Pour modifier un camouflage vous devez enregistrer en cliquant sur le dossier à droite dans la customisation

    Enregistrer votre cammouflage où vous le souhaitez

    Ouvrer un éditeur photo comme Photoshop

    Après avoir fini enregistrez le dans le dossier "UserSkin" (cf explications plus haute)

    Bravo vous avez desormais un cammouflage personalisé par vos soins !
